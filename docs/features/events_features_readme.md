# Event feature documentation bundle

This directory contains the initial documentation and metadata templates for event-like features.

## Contents

* `features_events.md`
  Generic documentation for event-feature storage and metadata conventions.

* `event_feature_template.json`
  Reusable JSON sidecar template for any event-like feature table.

* `word_features_example.tsv`
  Worked example of a word-level event feature table.

* `word_features_example.json`
  Worked example of a sidecar for word-level event features.

* `vowel_features_example.tsv`
  Worked example of a vowel-level event feature table.

* `vowel_features_example.json`
  Worked example of a sidecar for vowel-level event features.

## Intended use

These files are meant to be copied into the repository `docs/` area and used as the basis for implementation and public derivative documentation.
